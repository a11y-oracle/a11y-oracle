Dropdown Navigation Menu

This page demonstrates a WCAG-compliant dropdown navigation menu implementing the WAI-ARIA Menubar pattern.

Keyboard Interaction

Tab
Moves focus into and out of the menubar.
Left Arrow / Right Arrow
Moves focus between top-level menubar items.
Enter / Space
Opens the submenu (if present) and moves focus to the first item. Activates links.
Down Arrow
Opens the submenu (if closed) or moves to the next submenu item.
Up Arrow
Moves to the previous submenu item. Wraps to last item from first.
Escape
Closes the submenu and returns focus to the parent menubar button.
Home / End
Moves focus to the first/last item in the submenu or menubar.
