Color Contrast Test Fixtures

These elements test various color contrast scenarios for the visual-engine and axe-bridge libraries. Each demonstrates a different path through the incomplete color contrast resolution pipeline.

Gradient Backgrounds (Pixel Pipeline)

These elements have gradient backgrounds that axe-core cannot resolve. The visual-engine pixel pipeline captures a screenshot and analyzes luminance.

Dark gradient background with white text — should pass pixel analysis (all background pixels are dark, contrast is high).
White-to-black gradient with white text — should produce a split decision (passes against dark side, fails against light side).

CSS Halo — Text Stroke

These elements use -webkit-text-stroke to create a halo effect. The halo detector CSS fast-path checks stroke width and color contrast.

Low-contrast text with 1.5px black stroke on light bg — halo should pass (stroke color vs background has good contrast).
Low-contrast text with 0.5px stroke — halo should be rejected (stroke width below 1px minimum).

CSS Halo — Text Shadow

These elements use text-shadow for halo effects. Valid halos require 4+ directions with zero blur.

Low-contrast text with 4-direction zero-blur black shadow on solid bg — halo should pass (covers all quadrants, shadow color vs bg has good contrast).
Text with blurred shadow on gradient — halo rejected (blur > 0), falls through to pixel pipeline on gradient background.

Solid Background Controls

These elements have solid backgrounds. axe-core resolves these natively; they serve as controls to verify the bridge doesn't interfere.

Dark text on white background — standard passing contrast (~15.3:1).
Light gray text on white background — clear contrast failure (~1.6:1).

Large Text Thresholds

Large text (>= 24px or >= 18.66px bold) uses a reduced contrast threshold of 3.0:1 instead of the standard 4.5:1.

24px gray text on gradient — should pass with 3.0 large-text threshold (contrast ratio ~4.5:1 which fails normal but passes large).
18.66px bold gray text on gradient — also qualifies as large text, should pass with 3.0 threshold.

Background Image

Elements with background images trigger the complex-background path, skipping halo detection and going straight to the pixel pipeline.

White text on SVG background image — halo detector skips (complex bg), pixel pipeline captures actual rendered background.
