A11y-Oracle Test Sandbox

This sandbox contains WCAG-compliant UI patterns used as ground-truth fixtures for testing the A11y-Oracle speech engine.

Test Fixtures

  • Dropdown Navigation Menu — WAI-ARIA Menubar pattern with keyboard navigation
  • A11y Violations — Intentional violations for testing audit rules
  • Color Contrast Tests — Gradient, halo, and complex background contrast scenarios
  • Identical Links Same Purpose — URL normalization and link purpose comparison
  • Link in Text Block — Non-color visual differentiation for inline links
  • Target Size — Interactive element sizing and spacing
  • Scrollable Region Focusable — Keyboard accessibility of scrollable containers
  • Skip Link — Skip link visibility on focus
  • Aria Hidden Focus — Focusable elements inside aria-hidden containers
  • Focus Indicator — Visual focus indicator presence and visibility
  • Content on Hover — Hoverable and dismissible hover content
